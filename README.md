# Simple Quiz Application

A command-line quiz application built with Python that tests users with multiple-choice questions from different topics such as General Knowledge, Science, Mathematics, Geography, and Technology.

## Screenshot

![Quiz Application Screenshot](screenshots/quiz-demo.png)

## Features

* Multiple-choice quiz format
* Randomized question order for every session
* Input validation for user answers
* Real-time score tracking
* Percentage score calculation
* Performance evaluation and feedback
* Play-again option
* Graceful handling of keyboard interruption (Ctrl+C)
* Clean and modular Python code structure

## Technologies Used

* Python 3
* Built-in `random` module

## How to Run

### Clone the Repository

```bash
git clone https://github.com/faisalakhtar6180/Simple_Quiz_Application.git
```

### Navigate to the Project Folder

```bash
cd Simple_Quiz_Application
```

### Run the Program

```bash
python Simple_Quiz_Application.py
```
